# Algorithms & Data Structures

A collection of implementations of common **algorithms** and **data structures**, mostly written in **Python**.  
This repository is intended as a reference for learning, practicing, and reviewing fundamental computer science concepts.

## Structure

The repository is organized by **algorithm categories**.  
Each problem or implementation is placed inside the directory that corresponds to the algorithmic technique it uses.

Examples:

- `graph-algorithms/` – Graph-related algorithms and problems  
- `dynamic-programming-algorithms/` – Dynamic programming solutions  
- `data-structures-implementation/` – Implementations of core data structures  

This structure makes it easier to navigate the repository and find implementations based on the algorithmic paradigm.

## Contents

### Algorithms
Different algorithmic techniques are grouped by category, such as:

- Graph Algorithms  
- Dynamic Programming  
- Greedy Algorithms  
- Searching and Sorting  
- and other common algorithmic paradigms

### Data Structures
Core data structures are implemented in: `data-structures-implementation/`

This folder contains implementations of commonly used data structures for educational and practical reference.

## Language

Most implementations in this repository are written in **Python**.

## Purpose

This repository is mainly for:

- Practicing algorithmic problem solving
- Learning and reviewing data structures
- Keeping clean and categorized implementations
- Serving as a quick reference for common algorithms
